Installation
------------

   1. Prerequisites (platform independent)
   2. Prerequisites (Linux)
   3. Install instructions
   4. Run instructions



   1. Prerequisites (platform independent)

      ByWaf requires python 2.7 or 3.3 and python-concurrent.futures. It can be installed by:

      https://pypi.python.org/pypi/futures (lastest version)
      python setup.py install

   

   2. Prerequisites (Linux)

      ByWaf requires python 2.7 or 3.3 and python-concurrent.futures. It can be installed by:

      https://pypi.python.org/pypi/futures (lastest version)
      python setup.py install

	  
 
   3. Install instructions
   
      Obtain application code either by downloading an archive from https://github.com/depasonico/bywaf-owasp/ or cloning the GIT repo

	  Extrac and navigate to the bywaf source directory.


   4. Run instructions

      Simply run:

      python bywaf.py
